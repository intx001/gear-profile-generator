**Gear Profile Generator **
是一款基于 Web 端的高保真几何齿轮建模与 CAD 发生引擎。支持标准和径向变位直齿轮（Profile Shift）的数学建模，提供轻量化镂空连腔、1:1 物理割缝参数（Kerf）补偿以及高精度 1:1 SVG/DXF 矢量轮廓切片下载。内置交互式多齿联动副双轮啮合动画，是现代创客、机械结构发烧友、激光切割与 3D 打印爱好者的得力物理标定助手。

**Gear Profile Generator **
is a high-precision, web-based CAD modeler & involute gear profile engineer. It provides mathematical modeling for standard and profile-shifted gears with lightweight relief truss cutouts, 1:1 laser cut kerf width inflation compensation, and standard vector SVG layout downloads. Featuring a dynamic real-time multi-gear mesh and contact animation, it serves as an indispensable tool for makers, CNC machinists, and 3D printing enthusiasts.
